## ğŸ¯ What is Strategiz UI?

Strategiz UI is a modern, responsive web application that provides users with:

1. **Portfolio Management**: Track holdings across multiple exchanges and brokerages
2. **Strategy Development**: Create, backtest, and deploy algorithmic trading strategies
3. **Marketplace**: Discover and purchase pre-built trading strategies
4. **Educational Resources**: Learn trading concepts, technical analysis, and strategy development
5. **Real-time Market Data**: Live prices, charts, and market indicators
6. **Multi-Provider Integration**: Connect Coinbase, Binance, Kraken, Charles Schwab, and more

## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Installation
```bash
# Clone repository
git clone https://github.com/strategiz/strategiz-ui.git
cd strategiz-ui

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Start development server
npm start
```

Application will be available at `http://localhost:3000`

### Environment Variables
```env
REACT_APP_API_URL=https://api.strategiz.io
REACT_APP_TRADINGVIEW_KEY=your_tradingview_key
REACT_APP_GOOGLE_ANALYTICS_ID=UA-XXXXXXXXX-X
```

## ğŸ› Troubleshooting

### Common Issues

**Build fails with "out of memory"**
```bash
NODE_OPTIONS='--max-old-space-size=8192' npm run build
```

**Ports already in use**
```bash
lsof -ti:3000 | xargs kill -9  # Kill process on port 3000
```

**Environment variables not loading**
- Ensure `.env` file exists
- Restart development server
- Variables must start with `REACT_APP_`
